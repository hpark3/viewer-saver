# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of ViewerSaver seriously. If you believe you have found a security vulnerability, please report it to us as follows:

1. **Do not open a public issue.**
2. Send an email to the maintainer (contact info available on GitHub profile) or use the GitHub Private Vulnerability Reporting feature if enabled.
3. Provide a detailed description of the vulnerability and steps to reproduce it.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Security Guarantees

ViewerSaver is designed to run **locally**.
- All document captures happen on your machine.
- No data is uploaded to our servers.
- Playwright Chromium runs in a sandbox, but use caution with untrusted URLs.
